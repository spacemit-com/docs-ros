# Introduction

SpacemiT Robot SDK is designed for RISC-V robotic platforms and supports the SpacemiT K series chips. It covers:

- System and solution overviews
- Basic applications, such as sensors and multimedia
- Advanced development topics, including model quantization and performance optimization
- Platform indexes and examples

The SDK aims to provide reference materials for robot application development and integration on K series platforms.
