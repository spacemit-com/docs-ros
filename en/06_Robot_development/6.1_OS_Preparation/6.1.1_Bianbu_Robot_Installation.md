# 6.1.1 Bianbu Robot SDK Installation

```
Last Version: 11/09/2025
```

The **Bianbu Robot SDK** is **pre-installed** in the official custom firmware.
By default, this firmware already includes **ROS 2** and all required runtime environments, so **no additional installation is needed**.

If you are using **bianbu-desktop** or any other general-purpose system image provided by SpacemiT:

1. **Install ROS 2 first** â€“ see [Section 6.1.2](6.1.2_ROS2_Installation.md).
2. **Install the Bianbu Robot SDK runtime** using the following commands:

   ```bash
   sudo apt update && sudo apt install -y wget && cd ~
   wget https://archive.spacemit.com/ros2/bros.tar.gz
   cd /opt && sudo tar xzvf ~/bros.tar.gz
   ```

After extraction, the **Bianbu Robot SDK binaries** will be available in the `/opt/bros` directory.
You can **run all functional components directly** from this location.
