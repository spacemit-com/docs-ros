### 2.1 一键编译
```
cd components/model_zoo/vision
mm
```

### 2.2 快速运行
下载模型资源：
```
bash examples/yolov8/scripts/download_models.sh

bash scripts/download_assets.sh

```
运行检测：
```
yolov8 examples/yolov8/config/yolov8.yaml
```

（可选）使用摄像头进行实时检测：
```
yolov8 examples/yolov8/config/yolov8.yaml --use-camera --camera-id 0  # 请确认摄像头 ID
```

### 3.2 快速运行

下载模型资源：
```
mkdir -p ~/.cache/models/llm
cd ~/.cache/models/llm
# 示例下载qwen2.5 0.5b的模型
wget https://archive.spacemit.com/spacemit-ai/model_zoo/llm/qwen2.5-0.5b-instruct-q4_0.gguf
```
启动 LLM 服务：
```
 llama-server -m ~/.cache/models/llm/qwen2.5-0.5b-instruct-q4_0.gguf -t 8 --port 8080
```
启动语音对话：
```
voice_chat --llm-url http://localhost:8080 --model qwen2.5:7b --tts matcha:zh-en
```
## 4. Agent 智能控制

通过 Agent 框架可实现自然语言控制机器人。Agent 能够自动理解用户意图、调用工具链、执行操作并返回结果。本节以 Hermes Agent 为例，演示如何配置 Agent 并通过 MLink 工具网关控制设备。

### 4.1 安装与配置 Agent

Agent 支持 OpenClaw 和 Hermes Agent 两种方案，推荐使用 Hermes Agent（交互体验更佳）。

```bash
# 安装 Hermes Agent
sudo apt-get update && sudo apt-get install -y hermes-agent

# 配置模型（按向导配置密钥和模型）
hermes model

# 启动交互
hermes
```

### 4.2 配置 Robot Skills

Robot Skills 为 Agent 提供机器人领域的专业技能。安装后，Agent 可自动完成 SDK 获取、环境配置、编译构建、模型下载及性能测试等操作。

```bash
# 获取 Robot Skills
git clone https://github.com/spacemit-robotics/robot-skills.git

# 安装到 Hermes
mkdir -p ~/.hermes/skills/spacemit-robot
cp -rf robot-skills/skills/* ~/.hermes/skills/spacemit-robot/
```

验证安装结果：

```bash
hermes skills list
```

预期输出包含以下技能：

| 技能名称 | 技能组 | 来源 |
| :--- | :--- | :--- |
| spacemit-robot-llm | spacemit-robot | local |
| spacemit-robot-sdk-bootstrap | spacemit-robot | local |
| spacemit-robot-shared | spacemit-robot | local |

确认安装成功后，启动 Hermes 进行对话测试：

```bash
hermes
# 输入：请测试 spacemit llm 性能数据给我
```

Agent 将自动执行完整流程：获取 SDK 代码 → 配置环境 → 编译构建 → 下载模型 → 测试性能 → 返回结果。

更多机器人技能持续更新中，敬请关注。
