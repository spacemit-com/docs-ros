# V20260430

## 概括说明

V20260430 汇总 2026 年 4 月 K3 ROS / SDK 的主要更新。本阶段新增支持桌面机器人、轮式机器人、人型机器人和机械臂案例应用，并进一步丰富 SDK 的机器人感知、运动控制、定位导航和路径规划等基础能力。

本卷面向希望基于 K3 平台复现完整机器人方案、验证端侧 AI 能力和集成机器人基础功能包的开发者。用户可以从参考方案进入具体机器人形态，也可以按 AI 与算法、机器人开发、系统与平台等专题查找对应组件。

## 发布版本号

- **卷标识**：V20260430  
- **发布或冻结日期**：2026-04-30  
- **推荐 Git / SDK 标签或分支**：开源持续更新，GitHub / Gitee 同步发布；本卷对应最新提交日期为 2026-04-30，具体分支或标签以 manifest / repo 发布信息为准。  
- **关键 target 或产品配置**：K3 平台；具体 target 以产品工程配置和发布 manifest 为准。

## 更新内容

| 模块或领域 | 说明 |
| --- | --- |
| 桌面机器人 | 新增 K3 支持 Hugging Face 开源 Reachy mini 桌面机器人，支持纯本地语音对话、语音控制、目标跟踪和人脸跟踪。 |
| 机械臂应用 | 新增 K3 支持 Hugging Face 开源 LeRobot 机械臂，支持 hermes-agent、openclaw 启动控制动作。 |
| 轮式机器人 | 新增开源“知行”机器人方案，覆盖轮式底盘与机械臂组合，提供建图导航、运动控制和路径规划能力。 |
| 人型与四足机器人 | 新增支持宇树 R1 / G1、Go1 机械狗，以及上海人型青龙、北京人型天工等平台的端侧推理到仿真应用。 |
| 模型库 | 丰富模型库，新增声纹识别和强化学习模型示例。 |
| AI 网关 | 新增 AI 网关，对外提供 HTTP / WebSocket 接口，方便用户直接路由调用各模型能力。 |
| AI Lab | 新增可视化模型交互体验 AI Lab，支持云端在线体验模型能力和本地体验模型能力。 |
| 机器人基础能力 | 丰富机器人感知、运动控制、定位导航和路径规划基础能力，支撑更多机器人方案复现和二次开发。 |
| 文档体系 | 完善 K3 文档结构，覆盖平台概览、快速入门、参考方案、AI 与算法、机器人开发、系统与平台和版本说明。 |

**升级建议**：建议先同步最新 repo / manifest，并确认 target 和产品工程配置后重新构建 SDK。复现机器人案例时，应优先按对应参考方案准备硬件、模型、外设和运行脚本；涉及镜像、系统组件或外设驱动升级时，应保留上一版本镜像和工程配置，便于回滚验证。
