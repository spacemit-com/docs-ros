# V20260530

## 概括说明

V20260530 汇总 2026 年 5 月 K3 ROS / SDK 的主要更新。本阶段重点完善桌面机器人、轮式机器人、机械臂和人形机器人应用闭环，并增强端侧语言交互、声源定位、视觉推理、强化学习策略执行和导航控制能力。

本卷面向希望基于 K3 平台复现机器人应用、集成端侧 AI 能力和验证运动控制链路的开发者。用户可以从参考方案进入桌面机器人、轮式机器人、机械臂、人形机器人等应用，也可以按 AI 与算法、机器人开发、系统与平台等专题查找对应组件。

## 发布版本号

- **卷标识**：V20260530
- **发布或冻结日期**：2026-05-30
- **推荐 Git / SDK 标签或分支**：开源持续更新，GitHub / Gitee 同步发布；本卷对应最新提交日期为 2026-05-30，具体分支或标签以 manifest / repo 发布信息为准。
- **关键 target 或产品配置**：K3 平台；具体 target 以产品工程配置和发布 manifest 为准。

## 更新内容

| 模块或领域 | 说明 |
| --- | --- |
| 桌面机器人 | 完善 Reachy Mini 应用闭环，新增仿真控制、组件接口调试和一键启动应用，支持通过语音控制人脸跟踪与手势跟踪模式切换。 |
| 机械臂应用 | 完善 LeRobot 机械臂应用，支持接入 Hermes Agent，通过自然语言或工具调用触发机械臂执行任务。 |
| 轮式机器人 | 完善 Linksee 轮式机器人应用，新增语音控制能力，并修复 RGB-D 相机静态变换、电机反馈等异常。 |
| 人形机器人 | 完善人形机器人仿真与强化学习能力，支持前置策略链，并支持 MuJoCo 自动计算重力补偿。 |
| 模型库 | 丰富模型库能力：视觉统一推理接口并简化配置；LLM 优化 OpenAI Tool Calling 兼容；ASR 规范离线音频处理；TTS 默认使用量化模型并优化英文音素化；支持声纹注册。 |
| 声源定位 | 新增 DOA 声源定位组件，支持双声道和多声道 GCC-PHAT 定位、C++ / Python 接口和实时麦克风示例，并接入 Omni Agent 应用。 |
| AI 网关 | 完善 AI 网关，支持 ASR / TTS / VAD / Vision / LLM / Embed / Rerank 统一 HTTP / WebSocket API、模型按需加载、音频上传规范化、模型 MD5 / URL 元数据和前端控制台体验。 |
| 多媒体能力 | MPP 同步新版视频、VI、RTSP 能力；Audio 增强播放缓冲和流设备处理。 |
| 外设与控制 | IMU 支持 ICM42670P INT1 中断；电机支持参数设置、EtherCAT 周期任务优先级和 RoHS 空闲处理；IO 支持时间戳和中断事件。 |

**升级建议**：建议同步最新 repo / manifest 后重新构建 SDK，并按实际使用场景优先验证机器人参考方案、AI 网关、声源定位、音视频链路和外设控制链路。涉及 Linksee、Reachy Mini、LeRobot 或人形机器人方案时，应同步检查硬件连接、模型文件、launch 参数和运行脚本；涉及底盘或电机控制时，应重点验证电机反馈、静态变换、实时任务优先级和外设事件上报。
