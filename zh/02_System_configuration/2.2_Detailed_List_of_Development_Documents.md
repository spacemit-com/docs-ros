## Buildroot 开发指南 (BSP 与底层驱动)

**[Buildroot](https://www.spacemit.com/community/document/info?lang=zh&nodepath=software/SDK/buildroot/root_overview.md)**  
是适配 SpacemiT Stone 系列芯片的 BSP（Board Support Package）及 SDK，提供完整的嵌入式 Linux 构建能力，包含：

- OpenSBI（监管程序接口）
- U-Boot / UEFI（Bootloader）
- Linux Kernel
- RootFS（集成多种中间件与基础库）
- 示例程序与开发工具

其目标是为处理器提供完整 Linux 支持，帮助开发者快速开展驱动与底层系统开发。
